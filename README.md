# Is-QQ-in-group

让用户扫码检测 TA 是否在某个 QQ 群内

引用库 [qq-login-qrcode](https://github.com/qiangmouren/qq-login-qrcode) 本仓库只实现示例，具体逻辑请自行编写

# Example

```
git clone https://github.com/qiangmouren/Is-QQ-in-group
cd Is-QQ-in-group
npm i
npm run start
```

网页服务默认运行在 http://127.0.0.1:3000
